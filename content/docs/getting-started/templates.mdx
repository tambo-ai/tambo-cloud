# Templates

Hydra AI provides templates for common UI patterns.

## Recommended Templates

### Control Bar

The Hydra AI Control Bar lets users control your app using natural language. Simply press ⌘K or click the control bar to start controlling components conversationally.

<Cards>
  <Card
    icon={<TerminalIcon />}
    href="https://control-bar.usehydra.ai"
    title="Add the Control Bar"
  >
    Add natural language control to your React app with one command: ```bash npx
    hydra-ai-cli add control-bar ```
  </Card>
</Cards>

## Features

The Control Bar provides:

- Natural language component control
- Keyboard shortcut (⌘K) activation
- Conversational UI interactions
- Simple one-line installation
- Full React integration
