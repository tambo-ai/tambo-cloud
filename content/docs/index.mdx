## Introduction

Hydra AI is an AI-powered router that helps you build adaptive UIs by intelligently routing users to relevant features based on context and behavior.

## Key Features

- **Contextual Routing**: Direct users to relevant features based on behavior
- **Adaptive UI**: Surface UI components based on context
- **Simple Integration**: Easy to add to React apps
- **Type-Safe**: Built with TypeScript
- **Framework Agnostic**: Works with any TS frameworks

## Why Hydra AI?

Unlike static routers, Hydra AI adapts to user needs by:

- Learning from behavior patterns
- Adapting to context
- Optimizing discovery
- Reducing friction
