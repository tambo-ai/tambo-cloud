## How to Automate Slack Connect Invites

Below is a step-by-step guide on how to automate Slack connect invites in any React application.

## Accessing the Slack API

To access the Slack API, you'll need to create an app in the [Slack API Console](https://api.slack.com/apps).

Once you've created the app, you'll obtain your `SLACK_OAUTH_TOKEN`.

Your app will also need the following permissions:

- `channels:read`
- `channels:manage`
- `channels:write`
- `channels:write.invites`
- `conversations.connect:write`

<img src="/assets/slack-scopes.png" alt="Slack API Permissions" />

It will prompt you to install the app to your workspace. You will need to have admin access to the Slack workspace to do this.

## Setting Up the Project

> This guide uses Next.js but you only need to update the API file for other backend frameworks (examples below).

First, ensure you have a Next.js project ready. If not, create one:

```bash
npx create-next-app@latest slack-connect-automation
cd slack-connect-automation
npm install @hookform/resolvers zod react-hook-form
```

Add these environment variables to your `.env.local`:

```bash
SLACK_OAUTH_TOKEN=xoxb-your-token-here
INTERNAL_SLACK_USER_ID=your-slack-user-id
DISALLOWED_EMAIL_DOMAINS=gmail.com,yahoo.com
```

## Create a Slack API Utility File

This utility encapsulates Slack-specific logic and can be reused in any backend framework.

**`lib/slack.ts`**:

```typescript
import fetch from "node-fetch";
import { z } from "zod";

const SLACK_API_BASE = "https://slack.com/api";

const SlackAPIError = z.object({
  ok: z.literal(false),
  error: z.string(),
});

const SlackAPIResponse = z.object({
  ok: z.literal(true),
  channel: z.object({
    id: z.string(),
    name: z.string(),
  }),
  url: z.string().optional(),
});

type CreateSlackChannelResponse = z.infer<typeof SlackAPIResponse>;

async function callSlackAPI<T>(
  endpoint: string,
  body: Record<string, unknown>
): Promise<T> {
  const response = await fetch(`${SLACK_API_BASE}/${endpoint}`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      Authorization: `Bearer ${process.env.SLACK_OAUTH_TOKEN}`,
    },
    body: JSON.stringify(body),
  });

  const data = await response.json();

  if (!data.ok) {
    const error = SlackAPIError.parse(data);
    throw new Error(error.error);
  }

  return data as T;
}

export async function createSlackChannel(
  companyName: string,
  email: string
): Promise<CreateSlackChannelResponse> {
  // Validate email
  if (!email.match(/^[^\s@]+@[^\s@]+\.[^\s@]+$/)) {
    throw new Error("Invalid email format");
  }

  // Check email domain restrictions
  const disallowedDomains =
    process.env.DISALLOWED_EMAIL_DOMAINS?.split(",") || [];
  const emailDomain = email.split("@")[1];
  if (disallowedDomains.includes(emailDomain)) {
    throw new Error("Email domain not allowed");
  }

  const channelName = `hydra-${companyName.toLowerCase().replace(/\s+/g, "-")}`;
  const channelData = await callSlackAPI("conversations.create", {
    name: channelName,
    is_private: false,
  });

  const inviteData = await callSlackAPI("conversations.inviteShared", {
    channel: channelData.channel.id,
    emails: [email],
  });

  return {
    ...channelData,
    url: inviteData.url,
  };
}
```

## Next.js API Route

The API route leverages the utility file, allowing you to reuse the business logic elsewhere.

**`pages/api/slack.ts`**:

```typescript
import { NextApiRequest, NextApiResponse } from "next";
import { z } from "zod";
import { createSlackChannel } from "@/lib/slack";

const CreateSlackChannelSchema = z.object({
  companyName: z.string().min(1, "Company name is required"),
  email: z.string().email("Invalid email format"),
});

export default async function handler(
  req: NextApiRequest,
  res: NextApiResponse
) {
  if (req.method !== "POST") {
    return res.status(405).json({ error: "Method Not Allowed" });
  }

  try {
    const parsed = CreateSlackChannelSchema.parse(req.body);
    const slackResponse = await createSlackChannel(
      parsed.companyName,
      parsed.email
    );

    res.status(200).json({
      channelId: slackResponse.channel.id,
      channelName: slackResponse.channel.name,
      inviteLink: slackResponse.url,
    });
  } catch (error) {
    console.error("Error creating Slack channel:", error);
    res.status(400).json({ error: error.message });
  }
}
```

## React Component: Slack Channel Form

This form submits data to the Next.js API route but can be configured to target another backend (e.g., NestJS).

> Don't forget to use your UI components to make this look nice.

**`components/SlackChannelForm.tsx`**:

```typescript
"use client";

import { zodResolver } from "@hookform/resolvers/zod";
import { useForm } from "react-hook-form";
import { useState } from "react";
import { z } from "zod";

const CreateSlackChannelSchema = z.object({
  companyName: z.string().min(1, "Company name is required"),
  email: z.string().email("Invalid email address"),
});

type CreateSlackChannelInput = z.infer<typeof CreateSlackChannelSchema>;

export function SlackChannelForm() {
  const [isLoading, setIsLoading] = useState(false);
  const [inviteData, setInviteData] = useState<{
    channelId: string;
    channelName: string;
    inviteLink: string;
  } | null>(null);

  const form = useForm<CreateSlackChannelInput>({
    resolver: zodResolver(CreateSlackChannelSchema),
    defaultValues: { companyName: "", email: "" },
  });

  const onSubmit = async (values: CreateSlackChannelInput) => {
    setIsLoading(true);
    try {
      const response = await fetch("/api/slack", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify(values),
      });

      if (!response.ok) {
        throw new Error("Failed to create Slack channel");
      }

      const data = await response.json();
      setInviteData(data);
    } catch (error) {
      console.error("Error:", error);
    } finally {
      setIsLoading(false);
    }
  };

  if (inviteData) {
    return (
      <div>
        <h2>Channel Created</h2>
        <p>Channel Name: {inviteData.channelName}</p>
        <a
          href={inviteData.inviteLink}
          target="_blank"
          rel="noopener noreferrer"
        >
          Join Channel
        </a>
      </div>
    );
  }

  return (
    <form onSubmit={form.handleSubmit(onSubmit)} className="space-y-4">
      <div>
        <label>Company Name</label>
        <input
          {...form.register("companyName")}
          placeholder="Enter your company name"
          disabled={isLoading}
        />
      </div>
      <div>
        <label>Email</label>
        <input
          {...form.register("email")}
          type="email"
          placeholder="Enter your email"
          disabled={isLoading}
        />
      </div>
      <button type="submit" disabled={isLoading}>
        {isLoading ? "Creating Channel..." : "Create Slack Channel"}
      </button>
    </form>
  );
}
```

## Using a Different Backend

If you'd like to use a different backend, like NestJS or Express, the `lib/slack.ts` file can be easily imported. For example:

**NestJS Controller Example**:

```typescript
import { Controller, Post, Body, BadRequestException } from "@nestjs/common";
import { createSlackChannel } from "@/lib/slack";

@Controller("slack")
export class SlackController {
  @Post()
  async createChannel(@Body() body: { companyName: string; email: string }) {
    try {
      return await createSlackChannel(body.companyName, body.email);
    } catch (error) {
      throw new BadRequestException(error.message);
    }
  }
}
```
