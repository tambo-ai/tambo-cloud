## Getting Started

To get started with Tambo, follow these simple steps:

1. **Sign up** for a free account at [tambo.ai](https://tambo.ai)
2. **Create** your first project
3. **Deploy** your agent with one click

```bash
# Install the Tambo CLI
npm install -g @tambo-ai/cli

# Initialize a new project
tambo init my-agent

# Deploy to production
tambo deploy
```
