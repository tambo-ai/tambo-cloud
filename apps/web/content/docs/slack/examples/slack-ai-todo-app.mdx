This example is a Slack AI Todo app built with Hydra AI. This app allows you to conversationally interact with AI to create, update, and delete todo items, where the AI can respond with interactive UI components.

This can be used as a starting point for your own Slack AI app.

To see how this works step-by-step, check out the tutorial [here](/docs/slack/tutorials/slack-ai-todo-app/overview).
